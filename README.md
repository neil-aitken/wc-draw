# wc-draw

A new Python project managed with uv.

## Features
- Environment management with uv
- Testing with pytest
- Linting with ruff

## Setup

1. Install [uv](https://github.com/astral-sh/uv):
   ```powershell
   pip install uv
   ```
2. Install dependencies:
   ```powershell
   uv pip install -r requirements.txt
   ```
3. Run tests:
   ```powershell
   uv pip install -e .
   uv test
   ```
4. Lint code:
   ```powershell
   uv lint
   ```
